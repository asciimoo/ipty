IPTY
====

[Pseudo terminal](https://en.wikipedia.org/wiki/Pseudo_terminal) extension


![ipty.gif](docs/ipty.gif)


### Features

On-the-fly

 * Input completition
 * Input filter
 * Python evaluation


### Bugs

Bugs or suggestions? Visit the [issue tracker](https://github.com/asciimoo/ipty/issues).
